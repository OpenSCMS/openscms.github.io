Role of scmscommon

The scmscommon crate is a Rust library crate that contains functionality shared across multiple SCMS services. This includes common configuration handling, shared data types, error abstractions, and utility routines that are orthogonal to any specific SCMS role.

By centralizing this functionality in a dedicated crate, OpenSCMS avoids code duplication while maintaining strict service boundaries. Importantly, scmscommon does not implement SCMS business logic or protocol workflows, ensuring that each service crate remains responsible only for its own role as defined by the standard.
