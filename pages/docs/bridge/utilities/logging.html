Logging and Diagnostics

The library includes a logging and diagnostics framework to aid in debugging and operational monitoring.

Features

  • Structured logging: Log messages include context such as function name, file, and line number.
  • Log levels: Support for debug, info, warning, and error levels.
  • Integration with Rust: Logs generated in C can be forwarded to Rust's logging infrastructure.
  • Diagnostic utilities: Functions for dumping ASN.1 structures, cryptographic keys, and protocol messages in human-readable formats.

Logging can be configured at runtime to control verbosity and output destinations.
