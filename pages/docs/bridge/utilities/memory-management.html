Memory Management Utilities

The library provides a set of memory management utilities to ensure safe and consistent allocation and deallocation across the Rust-C boundary.

Key Utilities

  • Allocation wrappers: Custom allocation functions ensure compatibility between Rust and C memory models.
  • Cleanup routines: Functions for deallocating complex ASN.1 structures and cryptographic key material.
  • Memory pools: Optional pooling mechanisms for frequently allocated types.

All exported functions that allocate memory follow a consistent pattern: ownership is clearly documented, and corresponding cleanup functions are provided.
