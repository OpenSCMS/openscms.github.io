Integration with Rust

The C library is integrated into the Rust-based SCMS services using bindgen, which automatically generates Rust FFI bindings from the C header files.

Integration Approach

  • Automatic binding generation: bindgen parses the public C headers and generates corresponding Rust types and function declarations.
  • Safe wrappers: A thin Rust wrapper layer provides safe, idiomatic Rust interfaces over the raw FFI bindings.
  • Memory safety: Ownership and lifetimes are carefully managed to ensure no memory leaks or use-after-free bugs.
  • Error handling: C error codes are converted to Rust Result types for consistent error handling.

This integration approach allows Rust developers to work with the cryptographic and protocol functionality as if it were native Rust code, while benefiting from the performance and security characteristics of the C implementation.
