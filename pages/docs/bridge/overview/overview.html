Introduction

Overview

The oscms-codecs-bridge library is the low-level cryptographic and protocol core of OpenSCMS. It is implemented in C and provides all security-critical functionality, including cryptographic operations, ASN.1 encoding and decoding, and construction and verification of IEEE 1609.2.1 SPDUs.

This library is designed as a standalone component, independent of the Rust backend and SCMS service logic. Integration with the Rust-based services is performed transparently through Rust bindings generated using bindgen. From the perspective of a Rust developer, the library appears as a native Rust module, while all cryptographic and protocol complexity remains encapsulated in C.
