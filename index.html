About OpenSCMS

OpenSCMS is an open-source implementation of a Security Credential Management System (SCMS) compliant with IEEE 1609.2.1. The project focuses on providing a practical, modular, and deployable realization of the standard, covering its main architectural and functional components.

OpenSCMS implements the core elements defined by IEEE 1609.2.1 with an emphasis on efficiency and scalability, leveraging task parallelism to support high-throughput operation across different deployment sizes and scenarios.

A central part of the project is a low-level core library that implements cryptographic primitives and ASN.1 structure manipulation. This library is intentionally decoupled from the SCMS logic, allowing it to be reused independently, integrated into other SCMS implementations, or used in isolation for simulations, testing, and experimentation.

The project includes a complete deployment setup based on Docker and Kubernetes, enabling containerized service orchestration, monitoring, and lifecycle management. This supports both local testing environments (for example, using Minikube) and production deployments on servers or clusters.

OpenSCMS also provides auxiliary scripts for generating supporting artifacts such as CTL and CCF payloads, certificates, and cryptographic keys for electors and root CAs. Additional features, including logging, parameter customization, and fault recovery mechanisms, are included to improve robustness, observability, and operational flexibility.
